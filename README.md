# netSim Website

This folder contains a simple static website starter for `netSim`.

## Files

- `index.html`
- `styles.css`

## Local Preview

From this folder, you can preview it with a simple static server:

```bash
python3 -m http.server 8000
```

Then open:

```text
http://localhost:8000
```

## GitHub Pages

This structure is ready to be published as a simple static site on GitHub Pages.

Typical options:

- publish the repository root if this folder becomes its own repo
- or copy these files into a `docs/` folder inside a GitHub repository and publish from there
