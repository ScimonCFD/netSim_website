What is Angelica?

A simulator for steady-state pipe networks and accessories.

Angelica is built around the traditional workflow used in hydraulic network software: define sources and sinks, connect them through internal junctions, assign pipes and devices to the links, and run the case from a graphical interface.

The current solver supports pipes, fittings, pumps, elevation changes, Darcy-Weisbach/Colebrook-White, and Hazen-Williams. It also includes validated benchmark cases against EPANET-based references.

How is it used?

The main workflow is GUI-first.

01

Launch the GUI

Open the graphical interface and start a new network or load a validated tutorial case.

02

Build the network

Add sources, sinks, and junctions, then connect them with links and assign pipes, fittings, or pumps.

03

Configure the case

Define the material, pressure-drop model, and numerical settings from the GUI.

04

Run and inspect

Run the simulation, inspect convergence, and export converged results to a spreadsheet report.

Validation

Benchmark cases already included in the project.

Hanoi benchmark

A validated EPANET/Darcy-style Hanoi network case used to compare trunk-node heads and trunk-link flows.

EPANET pump benchmark

A pump-bearing benchmark adapted from the official EPANET tutorial network, including published reference pressures, heads, and flows.

Tutorial suite

Pipe-only, fittings, elevation, inlet-flow, outlet-flow, and combined-boundary cases are available for smoke testing and onboarding.
