Numerical Method

Modified SIMPLE algorithm for steady, incompressible pipe networks.

A pressure-correction strategy adapted from the SIMPLE method. The solver first builds a laminar solution — which converges in a single iteration for pure-pipe systems — then uses it as the initial condition for the nonlinear solve with turbulent and accessory element laws.

Solution sequence
  • Assume a pressure field P*
  • Estimate velocities from element laws
  • Assemble nodal correction system
  • Solve for P′ → update P and V
  • Converge laminar → seed nonlinear

Starting point

Three equations govern the solver.

Energy equation — steady flow in a pipe

Relates the pressure drop between two nodes to elevation and friction losses. This is the momentum equation the solver satisfies for each link.

$$(P_{in} - P_{out}) = \rho g\,(z_{out} - z_{in}) + f\,\frac{l}{D}\,\frac{\rho V^2}{2}$$

$D$ — internal diameter  ·  $l$ — pipe length  ·  $f$ — friction factor  ·  $\rho$ — fluid density  ·  $z$ — elevation  ·  $V$ — mean velocity

In laminar flow $\,f = 64/Re\,$ where $Re = \rho V D / \mu$, reducing the energy equation to the linear Poiseuille relation. In turbulent flow, $f$ is given by Colebrook-White.

Colebrook-White — turbulent friction factor

An implicit equation for $f$ given the Reynolds number and wall roughness. It is solved iteratively (or via an explicit approximation) at each iteration of the nonlinear stage.

$$\frac{1}{\sqrt{f}} = -2\log\!\left(\frac{\varepsilon/D}{3.7} + \frac{2.51}{Re\sqrt{f}}\right)$$

$\varepsilon$ — absolute wall roughness

Nodal continuity

Mass is conserved at every junction. The pressure-correction step is derived from this equation — its residual $b_C$ drives the iteration.

$$\sum_{in}\dot{m}_{in} \;=\; \sum_{out}\dot{m}_{out}$$

The SIMPLE decomposition

The solver assumes an initial pressure field $P^*$ and estimates velocities $V^*$. The true solution is related to the estimate by corrections $P'$ and $V'$:

$$P = P^* + P', \qquad V = V^* + V'$$

Substituting into the continuity equation and rearranging produces a linear system in $P'$. The central challenge — and what distinguishes element types — is writing $V'$ as a function of $P'$. For laminar flow this relation is exact. For nonlinear elements it requires an approximation.

Pressure-correction coefficients

How each element enters the nodal system.

Every element contributes a coefficient $a_f$ to the pressure-correction equation of each node it connects. The nodal system always takes the same form regardless of element type:

$$a_C\,P'_C \;+\; \sum_f a_f\,P'_F \;=\; b_C$$

where $\;a_C = -\displaystyle\sum_f a_f\;$ and $\;b_C = \displaystyle\sum_{in}\dot{m}^*_f - \sum_{out}\dot{m}^*_f$. What changes between element types is the expression for $a_f$.

Laminar pipe

The Poiseuille equation is linear in pressure, so the velocity correction $V'$ can be expressed exactly in terms of $P'$. The coefficient $c$ is constant for each pipe:

$$c = \frac{\rho A D^2}{32\,\mu\,l} \qquad \Longrightarrow \qquad a_f = -c_f$$

Because $a_f$ is constant, the pressure-correction system is linear and a network of purely laminar pipes converges in a single iteration. This is why the laminar stage is used to generate the initial condition for the nonlinear solve.

Turbulent pipe

The turbulent momentum equation is nonlinear in $V$, so an exact $V'(P')$ relation does not exist. Using the approximation $V \approx V^*$ — which holds exactly at convergence — gives:

$$V' = \frac{2D}{\rho\,l}\cdot\frac{P'_{in} - P'_{out}}{f_{V^*}|V^*|}$$

Substituting into the continuity equation yields the coefficient:

$$a_f = -\!\left(\frac{2AD}{l\,f_{V^*}|V^*|}\right)_{\!f}$$

$f_{V^*}$ is the friction factor at the current velocity estimate, updated every iteration via Colebrook-White. As $V^* \to V$, $f_{V^*} \to f$ and the approximation becomes exact.

Fitting (accessory)

The pressure drop across a fitting is quadratic in velocity, governed by a loss coefficient $K_L$. Applying the same approximation as the turbulent case:

$$\Delta P = K_L\,\frac{\rho V^2}{2} \qquad \Longrightarrow \qquad a_f = -\!\left(\frac{2A}{K_L|V^*|}\right)_{\!f}$$

$K_L$ values for elbows, valves, tees, and other standard fittings are available in the built-in library or can be entered as a custom coefficient.

Boundary conditions

Known pressure

Set $P' = 0$ at the boundary node. The correction is zero by definition — the prescribed pressure is already exact.

Known mass flow

The boundary flow is imposed directly. The $P'$ term for that node does not appear in the algebraic system — it is absorbed into $b_C$ of the connected interior nodes.

At least one pressure boundary is required for a unique solution. Mixed types — pressure sources, pressure sinks, flow inlets, flow outlets — can coexist in the same network.

The algorithm

Modified SIMPLE — laminar seed, nonlinear solve.

The SIMPLE loop runs in two stages. The first assumes laminar flow everywhere and converges rapidly. The converged laminar field seeds the second stage, which applies the correct nonlinear element laws — significantly reducing the iterations required.

Stage 1 — Laminar
Assume an initial pressure field $P^*$
↓
Estimate link velocities $V^*$ from the laminar momentum equation Pipe: Poiseuille  ·  Accessory: KL law
↓
Update mass flows   $\dot{m} = \rho V^* A$
↓
Assemble the pressure-correction system Element coefficient: laminar $c$  ·  Boundary conditions: BC rules
↓
Solve for $P'$  ·  update $P = P^* + P'$,  $V = V^* + V'$ SIMPLE decomposition
↓
Converged?  →  proceed to Stage 2 Pure-pipe systems converge in one iteration
Stage 2 — Nonlinear
Use the converged laminar field as initial $P^*$, $V^*$
↓
Recompute $V^*$ with the nonlinear element laws Turbulent pipe: Darcy-Weisbach + Colebrook  ·  Accessory: KL law
↓
Update mass flows   $\dot{m} = \rho V^* A$
↓
Assemble with nonlinear coefficients turbulent $a_f$  ·  accessory $a_f$  ·  BC rules
↓
Solve for $P'$  ·  update $P$ and $V$
↓
Converged?  →  solution complete Convergence criterion: nodal mass imbalance $b_C \to 0$
