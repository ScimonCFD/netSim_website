Numerical Method

Pressure-correction workflow for the steady, incompressible, isothermal solver.

Angelica uses a pressure-correction strategy that first builds a laminar solution and then uses that field as the initial condition for the nonlinear solve. The local pressure-correction relation is built differently for pipes, accessories, and pumps, but all of them enter the same nodal continuity framework.

Solver family
  • Assume pressure field
  • Estimate link velocities
  • Assemble pressure correction
  • Correct pressures and flows
  • Converge laminar field
  • Use it to start nonlinear solve

Workflow

High-level solution sequence.

Laminar stage
Assume an initial pressure field p*
↓
Estimate link velocities using the laminar element equations
↓
Update mass flows with m = ρ A v
↓
Assemble the pressure-correction system Laminar pipe coefficients from the linear pressure-flow relation
↓
Solve pressure correction and update pressure and velocity
↓
Repeat until laminar convergence
Nonlinear stage
Use the converged laminar field as the initial condition
↓
Recompute link velocities with the nonlinear element laws Turbulent pipes, accessories, and pumps
↓
Update mass flows
↓
Assemble the nonlinear pressure-correction system Each element contributes its own local pressure-correction coefficient
↓
Solve, correct, and iterate
↓
Stop when the field converges

Element laws

How each component enters the correction equation.

Laminar pipes

In laminar flow, the pressure-flow relation is linear. That makes it possible to generate an exact local pressure-correction relation for the pipe.

Turbulent pipes

In turbulent flow, the pressure-flow relation is nonlinear, so the local pressure-correction relation is constructed from the current state and updated during the iteration process.

Accessories

Accessories are treated through local-loss relations. Their own pressure-flow expression produces the local coefficient used in the system assembly.

Pumps

Pumps are treated as pressure-changing devices. The assumed pressure difference implies a required head, the pump curve gives the corresponding flow, and the local derivative supplies the correction coupling.

Interpretation

One continuity framework, different local laws.

The key idea is that the global nodal system always comes from continuity. What changes from one link type to another is the way the local pressure-correction contribution is generated.

In practice, pipes, accessories, and pumps all enter the same assembly process once their local relation between pressure difference and flow has been linearized around the current state.
