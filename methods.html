Numerical Method

The hydraulic loop at the core of Angelica.

Pressure-correction adapted from the SIMPLE method (Patankar, 1980), applied to networks of pipes, fittings, and pumps.

Governing equations

Two equations govern every pipe network.

Momentum — energy equation per pipe

Relates the pressure drop between two nodes to elevation change and friction losses along each link.

$$(P_{in} - P_{out}) = \rho g\,(z_{out} - z_{in}) + f\,\frac{l}{D}\,\frac{\rho V^2}{2}$$

$D$ — internal diameter  ·  $l$ — pipe length  ·  $f$ — friction factor  ·  $\rho$ — fluid density  ·  $z$ — elevation  ·  $V$ — mean velocity

In laminar flow $f = 64/Re$ where $Re = \rho V D / \mu$, reducing this to the linear Poiseuille relation. In turbulent flow, $f$ is a function of $Re$ and the relative wall roughness $\varepsilon/D$ — evaluated at each iteration via a closure relation (Colebrook-White).

Continuity — mass conservation at each node

Mass is conserved at every junction. The pressure field is driven by this equation — the pressure-correction algorithm is derived directly from it, and its residual $b_C$ measures how far the current iterate is from the solution.

$$\sum_{in}\dot{m}_{in} \;=\; \sum_{out}\dot{m}_{out}$$

These two equations are solved together. Even though Angelica uses a segregated pressure-correction algorithm, the converged solution satisfies both simultaneously at every node and every link.

The algorithm

Modified SIMPLE — laminar seed, nonlinear solve.

The SIMPLE loop runs in two stages. Stage 1 assumes laminar flow everywhere and converges in a single iteration for pure-pipe networks — producing an exact initial field. Stage 2 uses that field to seed the nonlinear solve with Darcy-Weisbach friction, Colebrook-White closure, fittings, and pumps.

Calculating V*

In this implementation, the provisional velocity field $V^*$ is computed from the current pressure field according to the active element law. The form of $V^*$ therefore depends on the regime and on the constitutive equation selected for each element type.

Laminar pipes

In Stage 1, pipe velocity is obtained from the linear laminar relation, including the elevation term. This is the Poiseuille-style form used to build the exact laminar seed.

$$V^* = \frac{D^2}{32\mu L}\left(\Delta P - \rho g \Delta z\right)$$

Turbulent pipes

In Stage 2, $V^*$ depends on the turbulent equation selected by the user. When Colebrook-White is selected, the Darcy-Weisbach relation is closed through the Colebrook friction factor. If another model is selected, such as Hazen-Williams, $V^*$ is evaluated using that selected relation instead.

$$\Delta P - \rho g \Delta z = \frac{\rho f L}{2D}\,V\,|V|$$

Fittings and local-loss devices

For fittings, $V^*$ is computed from the local-loss relation using the fitting loss coefficient $K_L$. The preset fitting coefficients currently included in Angelica follow the values proposed in Munson.

$$V^* = \operatorname{sign}(\Delta P)\sqrt{\frac{2|\Delta P|}{K_L \rho}}$$

Pumps

For pumps, the current pressure difference is converted into the head the pump must supply. The pump curve then returns the corresponding flow rate, which is converted to $V^*$ through the element area.

$$H_{\mathrm{req}} = -\frac{\Delta P}{\rho g}, \qquad Q = G(H_{\mathrm{req}}), \qquad V^* = \frac{Q}{A}$$

Pressure-correction derivation

How continuity becomes a correction equation.

The SIMPLE decomposition

The solver assumes an initial pressure field $P^*$ and estimates velocities $V^*$. The true solution is related to the estimate by corrections $P'$ and $V'$:

$$P = P^* + P', \qquad V = V^* + V'$$

Substituting into the continuity equation and rearranging produces a linear system in $P'$. The key step — and what distinguishes element types — is expressing $V'$ as a function of $P'$. For laminar flow this relation is exact. For nonlinear elements it requires an approximation that holds exactly at convergence.

The nodal pressure-correction system

Every element contributes a coefficient $a_f$ to the pressure-correction equation of each node it connects. The nodal system takes the same form regardless of element type:

$$a_C\,P'_C \;+\; \sum_f a_f\,P'_F \;=\; b_C$$

where $\;a_C = -\displaystyle\sum_f a_f\;$ and $\;b_C = \displaystyle\sum_{in}\dot{m}^*_f - \sum_{out}\dot{m}^*_f$. What changes between element types is the expression for $a_f$.

Laminar pipe — exact coefficient

The Poiseuille equation is linear in pressure, so $V'$ can be expressed exactly in terms of $P'$. The coefficient $c$ is constant for each pipe:

$$c = \frac{\rho A D^2}{32\,\mu\,l} \qquad \Longrightarrow \qquad a_f = -c_f$$

Because $a_f$ is constant, the pressure-correction system is linear and a network of purely laminar pipes converges in a single iteration.

Turbulent pipe — linearised coefficient

The turbulent momentum equation is nonlinear in $V$, so an exact $V'(P')$ relation does not exist. Using $V \approx V^*$ — which holds exactly at convergence:

$$a_f = -\!\left(\frac{2AD}{l\,f_{V^*}|V^*|}\right)_{\!f}$$

$f_{V^*}$ is the Colebrook-White friction factor evaluated at $V^*$, updated every iteration. As $V^* \to V$, the approximation becomes exact.

Fitting — loss coefficient

The pressure drop across a fitting is quadratic in velocity, governed by $K_L$. Applying the same linearisation:

$$\Delta P = K_L\,\frac{\rho V^2}{2} \qquad \Longrightarrow \qquad a_f = -\!\left(\frac{2A}{K_L|V^*|}\right)_{\!f}$$

Pump — curve-driven pressure changer

Pumps are treated as active pressure-changing devices. The current nodal pressure field defines the head the pump must supply:

$$H_{\mathrm{req}} = -\frac{\Delta P}{\rho g}$$

The pump curve then returns the corresponding flow rate, written locally as $Q = G(H_{\mathrm{req}})$. Angelica supports both single-point EPANET-style pump definitions and piecewise-linear multi-point $Q$-Head tables.

$$Q = G(H_{\mathrm{req}}), \qquad V = \frac{Q}{A}$$

For pressure correction, the pump contributes the local slope of the curve. This plays the same role as the hydraulic conductance of a pipe or fitting:

$$a_f \propto \frac{dQ}{d(\Delta P)} = \frac{dQ}{dH}\,\frac{1}{\rho g}$$

In other words, a pump enters the same nodal continuity system as every other element. What changes is only the local constitutive law used to relate pressure difference and flow.

Boundary conditions

Known pressure

Set $P' = 0$ at the boundary node. The prescribed pressure is already exact — no correction needed.

Known mass flow

The boundary flow is imposed directly. The $P'$ term for that node does not appear in the system — it is absorbed into $b_C$ of the connected interior nodes.

At least one pressure boundary is required for a unique solution. Mixed types — pressure sources, pressure sinks, flow inlets, flow outlets — can coexist in the same network.
